- Multiple sections: Hero, About, Skills, Projects, Activity, and Contact
- Fully responsive layout
- Custom CSS animations and effects

## 🚀 Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm start

# Build for production
npm run build
```

The site will open at `http://localhost:3000`

## 🎨 Customization

### Update Your Info
1. Edit component files in `src/components/`
2. Update styles in `src/styles/`
3. Modify colors in CSS variables

### Add Your Projects
Edit `src/components/Projects.js` to add your own projects.

### Change Social Links
Update URLs in `Hero.js` and `Footer.js` components.
